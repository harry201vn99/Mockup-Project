Enhancements

  • Align Images Side By Side
  1. Link:https://www.w3schools.com/howto/howto_css_images_side_by_side.asp
  2. This make the images side by side and centered in the web page, this technique was used in homepage (index page) and product page. This technique was applied by creating 2 divs: "row" and "column" div row covered all images and align them in centered position, while div column covered single images and make images run from left to right with changeable distance between each images.
  • Centered Top Navigation
  1. Link:https://www.w3schools.com/howto/howto_css_topnav_centered.asp
  2. This make the top navigation has centered text or image. This technique was applied in every single pages. This was created by creating span nav to cover all navigation links, span nav-centered to cover center image and span nav-right to cover right aligned link, while left aligned link was a default postion, so there was no need to create it. Along with some effect like current active web page and hover effect.
  • Image Hover Overlay - Fade In A Box
  1. Link:https://www.w3schools.com/howto/howto_css_image_overlay.asp
  2. This create a hover effect which appears text box and blurr effect on background image when mouse moved over. This technique was applied only in homepage (index page). This was created by creating 3 divs: container (align the overall effect on image), middle (change the text box from size to position) and text (change text size, family font, position).
  • Tables With Irregular Headers
  1. Link:https://www.w3.org/WAI/tutorials/tables/irregular/
  2. This make a table has headers in both row and column. This technique was applied in product page and about page. This was created by using scope row and scope col to make headers.
  • Form On Image
  1. Link:https://www.w3schools.com/howto/howto_css_form_on_image.asp
  2. This create a form of login with image as background. This technique was used in enquire page. This technique was created by using 3 divs: bg-img (apply background image to the form also change size, position of it), formlogin (change background color, text size and position) and formlog.
  • Rounded Images
  1. Link:https://www.w3schools.com/howto/howto_css_rounded_images.asp
  2. This make image rounded. This was applied in about page. By using border-radius in CSS applied at targeted image which set at 50%.
